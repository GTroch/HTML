1. Getting Started with Java

Installation & Setup

Before you start coding, you need to download and install the Java Development Kit (JDK).

Steps to Install:
  1. Download JDK from Oracle's official website
  2. Run the installer and follow the instructions
  3. Set up the JAVA_HOME environment variable
  4. Verify installation by opening Command Prompt and typing: java -version

2. Java Basics

Variables and Data Types

Variables are containers that store values. Java is a strongly typed language, so every variable must have a data type.

Data Type Size Description Example
int 4 bytes Whole numbers int age = 25;
double 8 bytes Decimal numbers double height = 5.9;
String Variable Text String name = "John";
boolean 1 bit True or False boolean isStudent = true;
char 2 bytes Single character char grade = 'A';

Variable Declaration Example

public class Variables { public static void main(String[] args) { int age = 20; String name = "Alice"; double gpa = 3.85; boolean isActive = true; System.out.println("Name: " + name); System.out.println("Age: " + age); } }

Operators

  • Arithmetic: +, -, *, /, %
  • Comparison: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !
  • Assignment: =, +=, -=, *=, /=

3. Control Flow Statements

If-Else Statement

Use if-else to make decisions in your program.

int score = 85; if (score >= 90) { System.out.println("Grade: A"); } else if (score >= 80) { System.out.println("Grade: B"); } else if (score >= 70) { System.out.println("Grade: C"); } else { System.out.println("Grade: F"); }

Switch Statement

int day = 3; String dayName; switch (day) { case 1: dayName = "Monday"; break; case 2: dayName = "Tuesday"; break; case 3: dayName = "Wednesday"; break; default: dayName = "Unknown"; }

Loops

For Loop

// Print numbers 1 to 10 for (int i = 1; i <= 10; i++) { System.out.println(i); }

While Loop

int count = 1; while (count <= 5) { System.out.println("Count: " + count); count++; }

Do-While Loop

int num = 1; do { System.out.println(num); num++; } while (num <= 5);

4. Methods (Functions)

Methods are reusable blocks of code that perform specific tasks.

Method Structure

public static <return_type> methodName(parameters) { // Method body return value; }

Examples

Method with No Parameters

public static void greet() { System.out.println("Hello!"); } public static void main(String[] args) { greet(); // Calling the method }

Method with Parameters and Return Value

public static int add(int a, int b) { return a + b; } public static void main(String[] args) { int result = add(5, 3); System.out.println("Sum: " + result); }

Overloading Methods

Methods can have the same name but different parameters.

public static int multiply(int a, int b) { return a * b; } public static double multiply(double a, double b) { return a * b; }

5. Arrays and Collections

Arrays

Arrays store multiple values of the same type in a single variable.

Array Declaration and Initialization

// Method 1: Declare and initialize with values int[] numbers = {10, 20, 30, 40}; // Method 2: Declare and initialize with size String[] names = new String[5]; // Access elements System.out.println(numbers[0]); // Output: 10 names[0] = "Alice";

Iterating Through Arrays

int[] scores = {85, 90, 78, 92}; // Using for loop for (int i = 0; i < scores.length; i++) { System.out.println(scores[i]); } // Using enhanced for loop for (int score : scores) { System.out.println(score); }

ArrayList

6. Object-Oriented Programming (OOP)

OOP is a programming paradigm based on objects and classes. It's fundamental to modern Java development.

Classes and Objects

A class is a blueprint for creating objects. An object is an instance of a class.

public class Student { // Attributes (Properties) private String name; private int age; private double gpa; // Constructor public Student(String name, int age, double gpa) { this.name = name; this.age = age; this.gpa = gpa; } // Getter and Setter methods public String getName() { return name; } public void setName(String name) { this.name = name; } // Regular method public void displayInfo() { System.out.println("Name: " + name + ", Age: " + age); } } // Creating objects Student student1 = new Student("Alice", 20, 3.85); student1.displayInfo();

The Four Pillars of OOP

1. Encapsulation

Bundling data (attributes) and methods into a single unit (class), hiding internal details from the outside world.

Benefits:
  • Data protection using access modifiers (private, public, protected)
  • Control over data access through getters and setters
  • Easy maintenance and modification

2. Inheritance

Creating a new class by inheriting properties and methods from an existing class.

public class Person { protected String name; public void greet() { System.out.println("Hello, I'm " + name); } } public class Teacher extends Person { private String subject; public Teacher(String name, String subject) { this.name = name; this.subject = subject; } public void teach() { System.out.println("Teaching " + subject); } }

3. Polymorphism

The ability to use an object in multiple forms or appear in many forms.

public class Animal { public void sound() { System.out.println("Some sound"); } } public class Dog extends Animal { @Override public void sound() { System.out.println("Woof!"); } } public class Cat extends Animal { @Override public void sound() { System.out.println("Meow!"); } }

4. Abstraction

Hiding complex implementation details and showing only the essential features.

abstract public class Vehicle { abstract public void start(); abstract public void stop(); } public class Car extends Vehicle { @Override public void start() { System.out.println("Car engine started"); } @Override public void stop() { System.out.println("Car stopped"); } }
